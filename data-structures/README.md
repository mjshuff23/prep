# Data Structures Practice

This folder is set up for scratch practice. Your existing implementations live next to new template files, so you can compare later without overwriting your work.

Suggested order:

1. `arrays/array_list_template.js`
2. `stacks/stack_template.js`
3. `queues/queue_template.js`
4. `linked-lists/singly_linked_list_template.js`
5. `linked-lists/doubly_linked_list_template.js`
6. `hash-tables/hash_table_template.js`
7. `sets/set_template.js`
8. `trees/binary_search_tree_template.js`
9. `trees/tree_traversals_template.js`
10. `heaps/min_heap_template.js`
11. `queues/priority_queue_template.js`
12. `graphs/graph_template.js`
13. `tries/trie_template.js`

How to use a template:

- Read the TODO comments first.
- Implement one method at a time.
- Add a few console checks at the bottom before moving on.
- Track `size` carefully. A lot of bugs hide there.
- Decide what empty operations should return before you code them.
